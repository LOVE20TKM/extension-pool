# 矿池类行动扩展协议

基于 LOVE20 扩展协议标准，扩展行动参与机制，允许以矿工和矿池主的方式参与矿池类行动。

## 1. 核心优势

### 1.1 参与体验提升

- 矿工：从概率铸币变为每轮确定铸币，参与门槛大幅降低，铸币体验大幅提升
- 规模：去中心化矿池机制支持现有社群加速裂变，外部社群大规模整体迁入

### 1.2 激励机制完善

- 矿池主：通过提供专业服务获得矿池主激励（详见[矿池主激励扩展协议](./矿池主激励扩展协议)）
- 市场化：竞争促进服务质量提升和专业团队形成

### 1.3 治理模式优化

- 效率：治理者从繁重验证工作中解放，聚焦行动决策和监督
- 公平：不信任投票机制保障验证公平性

## 2. 参与方式

### 2.1 矿工

通过[矿池扩展协议](./docs/矿池扩展协议.md)，参与一个正在运行的矿池，完成行动，并在矿池主的验证后，获得行动铸币激励

### 2.2 矿池主

在[矿池 ID](https://love20tkm.github.io/pool-id/)里铸造自己的矿池 ID，并在[矿池扩展协议](./docs/矿池扩展协议.md)中启动自己的矿池，为参与其中的矿工提供服务

在[矿池主激励扩展协议](./docs/矿池主激励扩展协议)里领取自己矿池对应的矿池主激励
