# 链群服务激励扩展协议（初稿）

## 1. 关于协议

此扩展协议，基于 LOVE20 扩展协议标准，扩展行动参与机制，对链群行动中的链群服务者做激励

- 链上参与：参与链群服务激励行动
- 自动验证：自动验证，并在验证阶段结束后铸造激励

## 2. 初始参数

- 代币地址
- 链群行动 id

## 3. 激励铸造

链群服务者每轮铸造激励，并同步销毁溢出的激励

链群服务者理论激励 = 链群服务者激励行动总激励 \* 链群服务者名下所有链群的实际可用票数 / 所有链群的实际可用票数

链群服务者实际激励 = 链群服务者理论激励 \* 容量修正系数

容量修正系数 = MIN（链群服务者链群理论容量上限 / 链群服务者名下所有链群已参与代币总量，1）

销毁激励 = 链群服务者理论激励 - 链群服务者实际激励
